# Project Flow Visualizer

Visualize your project structure as an interactive flow graph, similar to Unreal Engine's Blueprint editor.

## Features

- Interactive node-based visualization
- Collapsible folders
- File dependency tracking
- Vertical and horizontal layouts
- Zoom and pan controls

## Usage

1. Open a project folder in VS Code
2. Press `Ctrl+Shift+P` (or `Cmd+Shift+P` on Mac)
3. Type "Show Project Graph"
4. Enjoy your visual project structure!

## Controls

- **Click file** - Open in editor
- **Click folder** - Collapse/expand
- **Drag** - Pan around
- **Scroll** - Zoom in/out
